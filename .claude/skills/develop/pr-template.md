# PR テンプレート

draft PR の本文は以下の形式で作成すること:

```markdown
## Summary

<!-- 変更の目的と概要を1-3文で -->

## Changes

<!-- 変更内容を箇条書きで。各項目は具体的に -->

-

## Test plan

<!-- テストの実施内容 -->

- [ ] 既存テスト全パス (`make all`)
- [ ] 新規テスト追加: (テストファイル名)
- [ ] 手動確認: (確認手順があれば)

## Notes

<!-- レビュアーに伝えたいことがあれば -->

---
🤖 Generated with [Claude Code](https://claude.ai/code)
```
